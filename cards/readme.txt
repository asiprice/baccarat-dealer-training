Card Images Directory
=====================

To get high-quality playing card images:

OPTION 1 - Free Download (Recommended):
1. Go to: https://opengameart.org/content/playing-cards-vector-png
2. Download "Playing Cards.zip" (8.6 MB)
3. Extract the PNG files to this /cards directory
4. Rename files to match this format: [rank]_of_[suit].png

OPTION 2 - Alternative Sources:
- https://pngimg.com/image/8490 (free PNGs)
- https://www.vecteezy.com/free-png/playing-cards (free with account)

Required Filenames (52 cards total):
- ace_of_spades.png, ace_of_hearts.png, ace_of_diamonds.png, ace_of_clubs.png
- 2_of_spades.png, 2_of_hearts.png, 2_of_diamonds.png, 2_of_clubs.png
- 3_of_spades.png, 3_of_hearts.png, 3_of_diamonds.png, 3_of_clubs.png
- ... (continue pattern for 4,5,6,7,8,9,10) ...
- jack_of_spades.png, jack_of_hearts.png, jack_of_diamonds.png, jack_of_clubs.png
- queen_of_spades.png, queen_of_hearts.png, queen_of_diamonds.png, queen_of_clubs.png
- king_of_spades.png, king_of_hearts.png, king_of_diamonds.png, king_of_clubs.png

Notes:
- Images should be PNG format with transparent backgrounds
- Recommended size: 200x300 pixels or similar aspect ratio
- The game will automatically fallback to CSS-drawn cards if images aren't found
